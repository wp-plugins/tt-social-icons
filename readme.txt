== Description ==

TT Social Icons is a perfect WordPress Plugin for smooth operation of Social Icons by Widget in your WordPress Website. It is easy to use, customizable way to display social icons that link visitors to your various social profiles. You can easily choose which profiles to link, manage the size of the social icons with popular Social Networks. TT Social Icons has top and most popular 17 social icons.

<h3>Features</h3>

Manage the Size of the Social Icons
By default exact social color
Easily Customizable and managable code

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'TT Social Icons' Plugins in WordPress.
3. Go to Appearance => Click Widget => Give your social links in url field what you need, then Save.
